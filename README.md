rust-snake-ncurses
==================

A terminal-based snake game implementation in Rust using ncurses

## Dependencies

### Ubuntu

find what is the latest version of libncurses and install it

```sh
$ apt-cache search libncurses
$ sudo apt-get install libncurses5-dev -y
```

## Build & Run

```sh
$ cargo run
```
